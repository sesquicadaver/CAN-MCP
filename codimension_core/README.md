# codimension-core

Headless Python code analysis core for the Codimension fork.

Install from repo root (requires `codimension` package for parser fallbacks):

```shell
pip install -e .
pip install -e ./codimension_core
```

See [doc/CODIMENSION-CORE-MAP.md](../doc/CODIMENSION-CORE-MAP.md) for the extraction plan.
