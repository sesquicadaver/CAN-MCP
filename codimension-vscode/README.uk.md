# codimension-vscode

> **Мови:** [English](README.md) · [Українська](README.uk.md)

Базове розширення VS Code / Cursor для [Codimension MCP](../codimension_mcp/README.uk.md).

## Scope (0.4.0)

- Command palette → документація MCP
- **Codimension: Show Diagram (HTML WebView)** — `.codimension/diagrams/*.html` з `render_diagram`
- **Auto-open** при створенні HTML (налаштування `codimension.autoOpenDiagrams`)

## Збірка

```bash
cd codimension-vscode
npm install
npm run compile
```
